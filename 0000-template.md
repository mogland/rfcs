# Detailed design 详细设计

# Drawbacks 缺点

为什么我们不应该这样做呢?请考虑:

- 实现成本，包括代码大小和复杂性
- 集成此功能与其他现有和计划的功能
- NEXT 其他生态兼容的成本 （是 Breaking Change ？）

选择任何道路都需要权衡。试着在这里辨认他们。
